IVORY

A simple and modern CSS framework for developing powerful web interfaces faster and easier. It is a lightweight framework, just ~6KB. It gives you all the basic integrants to build your pages faster and easier.

Evolving the look and feel

IVORY is specially created for web applications and committed to improve the experience of users. It gives you modular and flexible components that are easily styled to build your pages faster and easier.

IVORY supports all the modern browsers and it is easy to customizable. You can customize and extend to get your web interface ready.
